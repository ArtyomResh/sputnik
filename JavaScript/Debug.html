Debug

Отладка

Во всех современных браузерах есть мощные инструменты разработчика, которые позволяют быстро и удобно находить ошибки в коде. Главное, научится ими пользоваться.

Пользуйтесь отладчиком, а не console.log.

В отладчике вашего браузера, можно задавать точки останова (breakpoints) для исполняемого кода. В тот момент, когда браузер дойдет до отмеченной строки, код будет остановление и у вас будет полный доступ к контексту исполнения: можно проверить значение всех переменных, запустить исполнение кода далее построчно, посмотреть текущий стек вызова. Всё это позволяет найти ошибку гораздо быстрее, чем обычное логирование значений. Get Started with Debugging JavaScript in Chrome DevTools
